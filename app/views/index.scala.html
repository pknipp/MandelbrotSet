Return to the Math APIs page.
creator:  Peter Knipp
repo:  https://github.com/pknipp/fractal

Background: The Mandelbrot set is an example of a fractal that arises from a simple mapping on the complex plane:
fc(z) = z2 + c The Mandelbrot set is the set of all complex numbers c for which infinite iterative calls of this function do not diverge. The boundary of this set is a fractal, in that it displays structure at all length-scales.

Controls:

  • Resolution: size of the hexagonally shaped cells into which the complex plane is divided
  • Iterations: number of times that the mapping is iterated before assuming that the mapping has not diverged
  • Magnification: number of times that the resolution has doubled
Instructions: After ...herokuapp.com above you should type
/⟨resolution⟩/⟨iter⟩/⟨mag⟩/⟨center⟩, in which the inputs are defined as follows:
  • resolution: (approximate) number of cells between the center and the edge of the viewing area
  • iter: number of times that the mapping is allowed to iterate before a determination is made whether the particular value of c belongs in the set
  • mag: number of times that the magnification has doubled
  • center: real and imaginary parts (aka coordinates) of the complex number at the center of the viewing area
